The interactive guide

This site offers guide to CSS 2.1 and 3, jQuery and HTML 5. You can start the interactive mode in each page by pushing START-button on the navigation bar. The mode is remembered when navigating the pages.

Instructions for the interactive mode

CSS

First write the selector as in CSS code. Then, write property name. Lastly, write the property value.

jQuery

Similarly to CSS, write first selector. Then write the method name. Lastly, parameters.
