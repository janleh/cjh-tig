jQuery

jQuery is the most popular JavaScript library. It includes over 500 functions. It is compatible with the following browsers:

  • Firefox 3.6+
  • Internet Explorer 6.0+
  • Safari 5.0+
  • Opera
  • Chrome

Functions and methods

The functions and methods in jQuery library can be divided in several ranges. The following table shows them giving destriction and explasen
