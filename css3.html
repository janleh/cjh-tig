CSS 3

CSS level 3 is the latest standard

New CSS 3 selectors

CSS selectors can be divided into following categories

  • Child selectors
  • Type selectors

Font freedom

CSS 3 introduced ability to display fonts that are not installed in the computer that views them.

New CSS properties

CSS 3 includes several new properties. These are:

  • Opacity
  • RGBA color
  • Text shadow
  • Rounded corners
  • Drop shadow
  • Border image
